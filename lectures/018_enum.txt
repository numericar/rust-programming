Enum (Enumeration)
    ใช้จัดเก็บข้อมูล ที่ถูกกำหนดคำตอบที่ถูกต้องเอาไว้อยู่แล้ว 
    โดยปกติจะใช้สำหรับจัดเก็บสถานะ

    เช่น 
        สัญญานไฟจราจร จะมี
            - เขียว
            - เหลือง
            - แดง
        แต่ทั้งหมด เราก็จะเรียกมันว่า ไฟจราจร

    ตัวอย่างการใช้งาน 
        enum CrabbyState {
            Resting,
            Fighting,
            Collecting(u32),
            Defending
        }
    จากตัวอย่าง Collecting สามารถรับข้อมูลได้ในรูปแบบ u32

    เราสามารถทำการเขียน function ให้กับ enum ได้ด้วย 
    impl CrabbyState {
        fn f(&self) {
            
        } 
    }
